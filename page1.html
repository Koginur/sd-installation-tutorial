Installation Automatic1111

Step 0: Before installation, be sure you have installed python 3.10.6 (Important note: when you installing python check "Add Python to PATH".) and git.

Okay, after you installed Python and git, you have to free 20-30gb space on your PC for SD.

Step 1: Create somewhere folder and name it "Stable_Diffusion".

Step 2: Open folder, press right mouse button and click Git Bash Here.



Step 3: You will see window like cmd, paste this command git clone https://github.com/AUTOMATIC1111/stable-diffusion-webui.git and press ENTER.



When process will successfully done, you can close git and we have folder named stable-diffusion-webui.

Congratulation! You have downloaded SD. But before opening it, you need checkpoint.

There is example of checkpoint you can use deliberate (checkpoint for realistic generation).

I will show how to download checkpoint Deliberate

After all this difficult steps you can start SD. Go to main folder and start bat file webui-user.bat.

You will see cmd its okay. For the first launch you will downloading some models for python.
